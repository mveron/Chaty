# Frontend

React desktop chatbot UI for the Chaty RAG backend.
UI generated with Google Stitch MCP and implemented as a desktop web workspace.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Runtime

Default backend URL is `http://localhost:8000` and can be changed from the Settings panel in the UI.
The layout is desktop-first (3-pane workspace).
